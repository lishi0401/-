1.2 详细引脚对照表

|       模块       | 接口     | STM32引脚        | 功能说明                     |
| :--------------: | -------- | ---------------- | ---------------------------- |
|  **RC522 RFID**  |          |                  | SPI接口                      |
|                  | SCK      | PA5 (SPI1_SCK)   | SPI时钟输出                  |
|                  | MOSI     | PA7 (SPI1_MOSI)  | SPI主出从入                  |
|                  | MISO     | PA6 (SPI1_MISO)  | SPI主入从出                  |
|                  | NSS      | PA4 (GPIO)       | SPI片选，低电平有效          |
|                  |          |                  |                              |
| **DHT11温湿度**  |          |                  | 单总线数字传感器             |
|                  | DATA     | PA8 (GPIO)       | 单总线数据口，需4.7K上拉电阻 |
|                  | VCC      | 3.3V/5V          | 供电                         |
|                  | GND      | GND              | 地                           |
|  **NEO-6M GPS**  |          |                  | 串口GPS模块                  |
|                  | TX       | PA3 (USART2_RX)  | GPS数据输出→MCU接收          |
|                  | RX       | PA2 (USART2_TX)  | MCU发送→GPS数据输入          |
|                  | VCC      | 3.3V/5V          | 供电（建议5V）               |
|                  | GND      | GND              | 地                           |
|   **串口Log**    | TX       | PA9              |                              |
|                  | RX       | PA10             |                              |
|                  |          |                  |                              |
| **MN316 4G模块** | TX       | PB10 (USART3_RX) | 4G数据输出→MCU接收           |
|                  | RX       | PB11 (USART3_TX) | MCU发送→4G数据输入           |
|                  | VCC      | 5V               | 模组工作电压（3.4-4.2V）     |
|                  | GND      | GND              | 地                           |
|     **OLED**     |          |                  | I2C接口显示屏                |
|                  | SCL      | PB8 (I2C1_SCL)   | I2C时钟                      |
|                  | SDA      | PB9 (I2C1_SDA)   | I2C数据                      |
|                  | VCC      | 5V/3.3V          | 供电                         |
|                  | GND      | GND              | 地                           |
|    **蜂鸣器**    |          |                  | 有源蜂鸣器                   |
|                  | I/O      | PA0 (GPIO)       | 低电平触发响                 |
|                  | VCC      | 5V               | 供电                         |
|  **LED指示灯**   |          |                  | 三色LED                      |
|                  | RED      | PA11 (GPIO)      | 温度/湿度报警（高亮）        |
|                  | GREEN    | PB14 (GPIO)      | 正常工作（高亮）             |
|                  | YELLOW   | PB13 (GPIO)      | 等待/录入中（高亮）          |
|                  | GND      | GND              | 共阴接法                     |
|   **按键输入**   |          |                  | 三个独立按键                 |
|                  | KEY1     | PB0 (GPIO)       | 模式切换/确认                |
|                  | KEY2     | PB1 (GPIO)       | 仓号加/上翻                  |
|                  | KEY3     | PB3 (GPIO)       | 仓号减/下翻                  |
|                  | GND      | GND              | 按键公共端                   |
|   **电源模块**   |          |                  | 12V车载电源转换              |
|                  | VIN      | 5v               | VBUS输入                     |
|                  | 5V OUT   | 5V               | 输出给4G模块、舵机           |
|                  | 3.3V OUT | 3.3V             | 输出给STM32、传感器          |
